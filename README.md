# 📂 Projetos da Faculdade

Este repositório reúne alguns projetos desenvolvidos durante meus estudos em **Engenharia de Software**.
Os projetos foram criados com o objetivo de praticar conceitos fundamentais de programação, análise de dados e organização de sistemas em Python.

Cada pasta contém um projeto independente com sua própria documentação.

---

# 🚀 Projetos

## 📊 Análise de Dados de Vendas

Projeto de **análise exploratória de dados (EDA)** utilizando Python para analisar informações de vendas armazenadas em um banco de dados SQLite.

Principais recursos:

* Criação de banco de dados SQLite
* Manipulação de dados com **Pandas**
* Visualização de dados com **Matplotlib e Seaborn**
* Identificação de padrões de vendas e insights

📁 Pasta do projeto:

```id="n8b94l"
/analise-de-dados
```

---

## 📝 Sistema de Gestão de Notas

Sistema simples desenvolvido em Python para **cadastrar notas de alunos**, calcular médias e exibir a situação final de cada estudante.

Principais recursos:

* Cadastro de alunos
* Registro de notas
* Cálculo automático de média
* Classificação em **Aprovado ou Reprovado**
* Resumo final da turma

📁 Pasta do projeto:

```id="kh1wd1"
/sistema-de-notas
```

---

## 📚 Sistema de Gerenciamento de Livros

Sistema em Python para **gerenciar um pequeno acervo de livros**, permitindo cadastro, busca e geração de gráficos por gênero.

Principais recursos:

* Cadastro de livros
* Listagem de livros
* Busca por título
* Gráfico de quantidade de livros por gênero
* Uso de **Programação Orientada a Objetos (POO)**

📁 Pasta do projeto:

```id="j2q1a3"
/sistema-livros
```
